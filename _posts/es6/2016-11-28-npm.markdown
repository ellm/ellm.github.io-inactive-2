## Common Commands

```bash
// List npm global packages
npm -g ls --depth 0

// Clear Cache
npm cache clean

// Uninstall global package
npm uninstall -g jscs

// Rebuild package
npm rebuild node-sass
```

- [Fix NPM permissions](https://docs.npmjs.com/getting-started/fixing-npm-permissions)
